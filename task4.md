## 4 (штраф). Документирование кода

[Doxygen](https://en.wikipedia.org/wiki/Doxygen#Example) - это специальный формат комментариев, который применяется, в частности, для документирования функций в C-коде.
Такой комментарий начинается с `/**` и внутри содержит специальные обозначения, среди которых
- `@brief`: краткое описание функции
- `@param`: описание параметра
- `@return`: описание возвращаемого значения

Пример:

```c
/**
 * @brief Calculates x raised to the power of pow
 * 
 * Computes the result of raising an integer base
 * to a non-negative integer exponent through iterative multiplication.
 * 
 * Example: power(2, 3) returns 8
 * 
 * @warning this function does not check for integer overflow
 * 
 * @param x   The base value
 * @param pow The exponent value
 * @returns x raised to the power of pow
 */
int power(int x, unsigned int pow)
{
    int result = 1;
    for(int i=0; i < pow; i++) result *= x;
    return result;
}
```

В данном задании необходимо написать функцию на языке Си и комментарии **на английском языке** к ней.

В каждом варианте
- ответвитесь от коммита с сообщением `add deadline`, сделанного ботом GitHub Classroom
- напишите функцию и комментарии к ней в файле `task3.c`
- в том же файле в функции `main` вызовите новую функцию
- влейте ветку `develop` в свою ветку
- влейте свою ветку обратно в `develop` через Pull Request
- обновите ветку `main`

## 4.1

Напишите функцию 
```c
int findSubarray(int* arr, unsigned int len, int target, unsigned int* start, unsigned int* end)
```
которая находит непрерывный подмассив целых чисел, сумма элементов которого равна `target`.
Функция возвращает 0, если такой подмассив найден, и 1 в противном случае.
При успешном поиске через указатели `start `и `end` возвращаются индексы начала и конца подмассива (включительно).
Если существует несколько подмассивов, подходит любой.

## 4.2
Напишите функцию 
```c
int mergeSortedArrays(int* arr1, unsigned int len1, int* arr2, unsigned int len2, int* result, unsigned int resultSize)
```
которая объединяет два отсортированных по возрастанию массива целых чисел в третий массив, также отсортированный по возрастанию.
Функция записывает результат в массив `result` (предполагается, что его размер достаточен) и возвращает 0 в случае успеха.
Если в процессе объединения обнаруживается, что результат не помещается в `resultSize` элементов, функция прекращает выполнение и возвращает 1.

## 4.3

Напишите функцию 

```c
int removeDuplicates(int* arr, unsigned int* len)
```

которая удаляет дубликаты из отсортированного по возрастанию массива целых чисел.
Функция модифицирует исходный массив так, что в начале располагаются уникальные элементы в порядке возрастания, и обновляет значение `len` так, чтобы оно соответствовало новой длине массива.
Функция возвращает количество удаленных элементов.

Например, для массива `[1, 1, 2, 2, 2, 3, 4, 4]` после вызова массив должен стать `[1, 2, 3, 4, ...]`, а `len` должно стать равным 4.

## 4.4

Напишите функцию 

```c
int rotateMatrix(int** matrix, unsigned int size, char direction)
```
которая поворачивает квадратную матрицу целых чисел размера `size` x `size` на 90 градусов.
Параметр `direction` может принимать значения 'L' (влево/против часовой стрелки) или 'R' (вправо/по часовой стрелке).
Функция модифицирует исходную матрицу и возвращает 0 в случае успеха.
Если направление указано неверно (какой-либо символ, кроме 'R' или 'L'), функция возвращает 1.
Матрица представлена как двумерный массив, выделенный в динамической памяти.
